# SelfieCoach

AI-powered selfie analysis to help you look your best.

## Website

This repository contains the professional marketing website for the SelfieCoach mobile application.

### Pages

- **index.html** - Main landing page showcasing the app features and benefits
- **support.html** - Customer support page with FAQ and contact form
- **privacy.html** - Comprehensive privacy policy page

### Features

- Responsive design using Bootstrap 5
- Professional gradient hero sections
- Interactive FAQ accordion
- Contact forms
- Mobile-friendly navigation
- Modern, clean design

### Viewing the Website

Open `index.html` in your web browser or serve the files using any web server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js
npx serve

# Then open http://localhost:8000 in your browser
```

### Technologies Used

- HTML5
- CSS3
- Bootstrap 5.3.2
- Bootstrap Icons
- JavaScript (for interactive elements)

## About SelfieCoach

SelfieCoach is a mobile application that uses AI to analyze selfies and provide personalized feedback on:

- Lighting quality and direction
- Photo composition and framing
- Focus and image clarity
- Facial expressions and engagement
- Background selection
- Color balance and skin tone

Upload your selfie and get professional-level tips to improve your appearance across various categories.
